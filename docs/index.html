readme

build

Generate README.md from README.tsx

build [--check] [--file ] [--output ]
FlagDescription
--checkCheck if the output is up to date (exit 1 if stale)
--file <file>TSX source to build (default: README.tsx in the target dir)
--output <output>Markdown output path (default: source with .tsx->.md)

docs

Generate docs/index.html command reference from .mise/tasks

docs [--name ] [--tagline ] [--repo ]
FlagDescription
--name <text>Project display name (default: target dir basename)
--tagline <text>One-line project tagline
--repo <url>GitHub repo URL (default: from git remote)

pre-commit:install

Install a git pre-commit hook that keeps README.md in sync

pre-commit:install [--build] [--check]
FlagDescription
--buildAuto-rebuild and stage README.md on commit
--checkError if README.md is stale on commit

pre-commit:remove

Remove the readme pre-commit hook

pre-commit:remove
